# LightVerse — Reading Platform (Data-aware UX)

## Overview
A responsive reading platform with features that support structured interaction signals (search, categories, bookmarking).

## Goal
Improve content discovery and reading experience so users can find relevant novels faster and continue reading smoothly.

## Tech Stack
- HTML
- CSS
- JavaScript

## What I Built
- User-friendly browsing and reading UI
- Search and category-based filtering
- Bookmarking and personalization-friendly UX patterns
- Responsive design + dark/light mode

## Next Steps (Data Analyst angle)
- Instrument feature usage (search success, bookmarks, retention signals).
- Build dashboards to compare discovery funnel performance across cohorts.
- Iterate with data-driven UX experiments (A/B tests or phased rollouts).
