Global Sugar Consumption Trends

A data analysis project using Microsoft Excel to explore worldwide sugar consumption patterns (1960–2023).

MS Excel Data Analysis Dashboards

1) Goal / Business Question

Explore worldwide sugar consumption patterns across regions, understand per capita usage, and forecast trends based on historical data.

2) Data

Type: Global and regional sugar consumption metrics spanning 1960 to 2023.

3) Approach (What I did)

  • Cleaned and structured historical dataset in Excel.
  • Built interactive pivot tables and charts.
  • Analyzed multi-decade trends to correlate with macroeconomic shifts.

4) Findings

  • Highlighted significant regional surges in consumption.
  • Identified steady upward trajectories and recent stabilization in specific markets.

5) Deliverables

  • Interactive Excel Dashboard for exploring trends natively.
  • Comprehensive summary report spanning 6 decades.

6) Next Steps

  • Cross-reference consumption trends with global health databases (e.g., obesity or diabetes rates).
